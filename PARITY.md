## Отсутствующие в мобильном приложении (9 экранов)

### Функции тренера
- **Посещаемость** (`Attendance`) — отметка посещаемости по группам
- **Создание внутреннего турнира** (`CreateInternalTournament`) — создание турнирной сетки
- **Детали внутреннего турнира** (`InternalTournamentDetail`) — просмотр/редактирование сетки

### Функции суперадмина
- **Добавление турнира** (`AddTournament`) — создание глобальных турниров
- **Добавление тренера** (`AddTrainer`) — создание тренеров вручную
- **Детали тренера** (`TrainerDetail`) — просмотр/редактирование тренера
- **Клубы** (`Clubs`) — управление клубами
- **Детали клуба** (`ClubDetail`) — просмотр/редактирование клуба

### Общие
- **Информация об авторе** (`Author`) — страница «О разработчике»

## Примечания

- Мобильное приложение покрывает основной пользовательский путь (ученик + тренер): вход, дашборд, команда, группы, турниры, финансы, материалы, профиль
- Функции суперадмина (управление тренерами, клубами, глобальными турнирами) доступны только через веб-приложение
- Внутренние турниры с турнирной сеткой (brackets) — сложная визуализация, реализована только в вебе
- Посещаемость — приоритетный кандидат для добавления в мобильное приложение
